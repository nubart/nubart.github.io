So testen Sie Nubart LIVE

Ihre Gruppe hört Sie klar und deutlich auf ihren eigenen Smartphones – mit optionaler KI-Übersetzung und synchronisierten Medien

Ihr Test umfasst

Eine Registrierung als Guide für den Test.
6 einzigartige QR-Karten zum Testen, abgedruckt auf der letzten Seite dieses Dokuments.
Eine Demo-Medienbibliothek, die der Guide live auf die Bildschirme aller Teilnehmer senden kann.
30 Minuten KI-Simultanübersetzung. Dies ist optional – Nubart LIVE funktioniert auch ohne diese Funktion einwandfrei.
Praktischer Tipp: Drucken Sie die vollständige Anleitung jetzt aus und schneiden Sie die sechs Karten aus, bevor Sie losgehen. Das macht den Test vor Ort deutlich einfacher.
Bevor Sie sich mit Ihrer Gruppe treffen
1

Registrieren Sie sich als Guide und erstellen Sie Ihr Passwort

Registrierungslink (verwenden Sie diesen einmalig, um Ihr Konto zu erstellen):

Ihre registrierte E-Mail-Adresse:
Mit dieser Adresse melden Sie sich an. Halten Sie sie bereit.

Anmeldelink (verwenden Sie diesen danach jedes Mal):

https://www.nub.art/customer/login
Tipp für den Testtag: Lesezeichen setzen www.nub.art/customer/login auf Ihrem Smartphone, damit Sie sich schnell anmelden können, wenn Sie mit Ihrer Gruppe zusammen sind.
2

Drucken Sie die 6 Teilnehmerkarten aus und schneiden Sie sie aus

Schneiden Sie die 6 Karten von der letzten gedruckten Seite aus – das dauert weniger als eine Minute. Halten Sie sie bereit, um sie zu verteilen, wenn Sie Ihre Gruppe treffen.

Jeder Code ist einzigartig und nicht übertragbar. Eine Karte pro Person – bitte nicht teilen oder tauschen. Schreiben Sie am besten den Namen jedes Teilnehmers auf die Karte, damit diese bei wiederholten Sitzungen dieselbe Karte behalten.
Durchführung des Tests – führen Sie diese Schritte mit Ihrer Gruppe durch
3

Melden Sie sich an und öffnen Sie die Gruppe

Rufen Sie auf Ihrem Smartphone www.nub.art/customer/login (oder verwenden Sie Ihr Lesezeichen) und melden Sie sich mit dem Passwort an, das Sie in Schritt 1 erstellt haben.

Tippen Sie nach der Anmeldung auf „Gruppe öffnen”.

KI-Übersetzung: Unmittelbar nach der Anmeldung erscheint ein Pop-up-Fenster, in dem Sie gefragt werden, ob Sie das Übersetzungs-Add-on aktivieren möchten. Sie können es hier aktivieren oder überspringen – Sie können dies später jederzeit ändern.
4

Scannen Sie die Teilnehmerkarten mit Ihrem Smartphone

Zuerst scannen Sie als Guide die Karten.

Tippen Sie auf dem Guide-Bildschirm auf „Scanner öffnen“ und scannen Sie die QR-Karten der Teilnehmer, die an Ihrem Test teilnehmen – Sie müssen nicht alle 6 scannen, wenn Sie nur mit einer kleineren Gruppe testen.

Wir empfehlen, die Karten vor dem Austeilen zu scannen – das ist der einfachste Weg. Allerdings können die Teilnehmer auch zuerst ihre eigene Karte scannen und sie Ihnen dann zeigen. Beides funktioniert problemlos.
5

Verteilen Sie die Karten – bitten Sie die Teilnehmer, ihre eigene Karte zu scannen

Anschließend scannt jeder Teilnehmer seine eigene Karte.

Geben Sie eine Karte pro Person aus. Jeder Teilnehmer scannt den QR-Code auf seiner eigenen Karte mit seinem Smartphone.

Tippen Sie auf Ihrem Guide-Bildschirm auf „Teilnehmer anzeigen“. Die grüne Anzeige neben jedem Platz leuchtet auf, sobald ein Teilnehmer die Verbindung von seinem eigenen Gerät aus hergestellt hat.

6

Sprechen Sie los. Probieren Sie die KI-Übersetzung aus, wenn Sie möchten

Tippen Sie auf „Stummschaltung aufheben“ und sprechen Sie. Auf dem Bildschirm sehen Sie einen sich bewegenden Balken – dies bestätigt, dass Ihre Stimme an alle Teilnehmer übertragen wird.

Ihr Test umfasst 30 Minuten kostenlose Übersetzung – probieren Sie diese gerne jetzt aus.

Wenn die Übersetzung aktiv ist: Die Teilnehmer erhalten sie automatisch in der Sprache ihres Browsers . Wichtig für den Test: Wenn die Telefone Ihrer Kollegen dieselbe Browsersprache wie Ihres haben, hören sie Ihre Originalstimme statt einer Übersetzung. Bitten Sie sie, auf das Symbol „ “ (Übersetzen) auf ihrem Bildschirm zu tippen, um manuell eine andere Sprache auszuwählen.
7

Probieren Sie die Medienbibliothek aus

Tippen Sie jederzeit während des Tests auf „Fernsteuerung“ auf Ihrem Guide-Bildschirm. Eine Beispiel- Medienbibliothek, die wir für diesen Test vorbereitet haben, wird geöffnet – sie ist voll funktionsfähig und einsatzbereit.

Tippen Sie auf einen beliebigen Eintrag in der Bibliothek, um ihn sofort an die Smartphones aller Teilnehmer zu senden. Dieser wird automatisch auf deren Bildschirm angezeigt – sie müssen nichts tun. Sie können weiterreden, während der Inhalt angezeigt wird.

So können Sie genau steuern, was Ihre Gruppe sieht – genau in dem Moment, in dem Sie es erklären.

Wenn Sie auf ein anderes Element tippen, ersetzt es das vorherige auf den Bildschirmen der Teilnehmer. Wenn Sie fertig sind, tippen Sie auf „Schließen”, um das Bibliotheksfenster auf Ihrem Bildschirm zu schließen.

Schaltflächenfarben in der Medienbibliothek: grün = noch nicht gesendet, grau = bereits gesendet, rot = Übertragung fehlgeschlagen (erneut antippen, um erneut zu senden).
In Ihrem Echtkonto: Die Medienbibliothek wird von unserem Team auf der Grundlage Ihrer Inhalte und Ihrer Tourroute erstellt. Sie kann Bilder, Bildstrecken, Audiospuren, Videos, 3D-Objekte, nebeneinander angeordnete Bilder und 360°-Panoramen enthalten. Wenn Sie mehrere verschiedene Routen betreiben, kann jede über eine eigene Bibliothek verfügen.
Nach dem Test
8

Lassen Sie die Gruppe offen oder schließen Sie sie

Wenn Sie später einen weiteren Test mit derselben Gruppe durchführen möchten, lassen Sie die Gruppe einfach offen. Wenn Sie sich das nächste Mal anmelden, sind Ihre Teilnehmer bereits verbunden – ein erneutes Scannen der Karten ist nicht erforderlich.

Wenn Sie mit dem Testen fertig sind, tippen Sie auf „Gruppe schließen“, um die Sitzung zu beenden.

Was die Teilnehmer sehen und tun können:
Sobald die Verbindung hergestellt ist, hören die Teilnehmer den Guide über ihr eigenes Smartphone. Wenn die Übersetzung aktiviert ist, erhalten sie diese automatisch in der Sprache ihres Browsers – oder sie können auf das Symbol „ “ tippen, um eine andere Sprache auszuwählen. Wenn der Guide ein Medienelement sendet, erscheint es automatisch auf ihrem Bildschirm – es sind keine weiteren Schritte erforderlich. Sie können dem Guide außerdem jederzeit eine schriftliche Frage senden, ohne ihn zu unterbrechen – der Guide sieht diese in einer Liste und kann antworten, wann es ihm passt.
Möchten Sie Nubart LIVE für Ihre eigenen Touren nutzen?
Wir richten Ihre erste Medienbibliothek ein und helfen Ihnen beim Einstieg. Antworten Sie einfach auf die E-Mail, in der Sie diesen Link erhalten haben.
Fehlerbehebung
Sie sind nicht stummgeschaltet, aber die Teilnehmer hören nichts

Überprüfen Sie, ob der bewegliche Balken (Spektrumanzeige) reagiert, wenn Sie sprechen. Falls nicht:

  • Möglicherweise nutzt eine andere App Ihr Mikrofon – schließen Sie alle Hintergrund-Apps und versuchen Sie es erneut
  • Wenn Sie ein Bluetooth- oder drahtloses Mikrofon verwenden, wechseln Sie zum Firefox-Browser, der externe Audiogeräte zuverlässiger verarbeitet als Chrome oder Safari
Die Teilnehmer sehen den Bildschirm, hören aber keinen Ton

Bitten Sie sie:

  • Erhöhen Sie die Lautstärke ihres Telefons
  • Überprüfen Sie, ob der Ton in der Teilnehmeroberfläche nicht stummgeschaltet ist
  • Schließen Sie alle anderen Apps, die möglicherweise Ton wiedergeben (Musikplayer, Videos usw.)
Die Übersetzung ergibt keinen Sinn

Dies wird fast immer durch eine schlechte Audioqualität verursacht. Überprüfen Sie die Audioqualitätsanzeige auf Ihrem Guide-Bildschirm :

  • Grün = Ton ist klar genug für die Übersetzung
  • Gelb oder rot = Tonqualität muss verbessert werden

So verbessern Sie die Qualität: Verwenden Sie ein Headset mit einem Mikrofon, das sich nahe an Ihrem Mund befindet, um Ihre Stimme von der Umgebung abzuschirmen, sprechen Sie deutlich und vergewissern Sie sich, dass die richtige Sprache ausgewählt ist.

Die Übersetzung wiederholt ständig Sätze

Ihr Mikrofon nimmt wahrscheinlich die übersetzte Audioausgabe von Geräten in der Nähe auf. Versuchen Sie Folgendes:

  • Verwenden Sie ein Headset mit Mikrofon
  • Bitten Sie die Teilnehmer, Kopfhörer zu verwenden
  • Erhöhen Sie den Abstand zwischen den Telefonen
  • Die Lautstärke bei Geräten in der Nähe verringern
Ein Teilnehmer erscheint nicht in der Teilnehmerliste

Möglicherweise hat er seine Karte noch nicht gescannt. Bitten Sie ihn, seine eigene QR-Karte erneut zu scannen – oder, falls er nicht scannen kann, auf nub.art zu gehen und den auf seiner Karte aufgedruckten privaten Code einzugeben. Die Verbindung kann nur vom eigenen Gerät des Teilnehmers aus hergestellt werden.

Die Verbindung bricht während des Tests ab

Dies ist in der Regel eine vorübergehende Lücke in der Netzabdeckung. Bewegen Sie sich einfach mit Ihrer Gruppe ein paar Meter weiter – die Verbindung wird automatisch wiederhergestellt, sobald wieder Netzabdeckung vorhanden ist.

Eine Schaltfläche der Medienbibliothek wird rot

Der Inhalt konnte nicht übertragen werden – meist aufgrund einer kurzen Verbindungsstörung. Tippen Sie einfach erneut auf die Schaltfläche, um ihn erneut zu senden. Wenn mehrere Schaltflächen gleichzeitig rot werden, überprüfen Sie, ob sowohl das Smartphone des Guides als auch die Smartphones der Teilnehmer über eine aktive Internetverbindung verfügen.
