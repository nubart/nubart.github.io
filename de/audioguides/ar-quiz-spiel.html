Gamification & Audioguide, alles in einem


Nubart® Game erweitert Ihren Nubart Audio Guide durch die Integration eines spannenden Gamification-Tools für jede Altersgruppe.

Lassen Sie Ihre Besucher spielen und lernen durch:

  •  Anhören einer Tonspur des mobilen Nubart Audioguides
  •  Fangen eines AR-Widgets in der Nähe der besprochenen Station, und / oder
  •  Beantwortung einer Frage im Zusammenhang mit dem soeben Gehörten.

Da unsere Audioguide-Karten einen individuellen, nicht übertragbaren Code enthalten, wissen Sie immer genau, wer gewonnen hat. Sie können selbst entscheiden, was der Preis sein soll und ihn an der Rezeption, im Shop oder im Café abholen lassen.

Die Ergebnisse des Spiels können Material für die Veröffentlichung in sozialen Medien generieren und so die Bindung zu Ihrem Publikum stärken.

Und das Beste daran: Sie werden kaum eine vollständig personalisierbare Gamification-App finden, die so erschwinglich ist wie Nubart Game.

Warum Nubart Game?



Weil es immer wichtiger wird, Interaktivität in Museen und anderen kulturellen Einrichtungen anzubieten, um ein junges Publikum anzuziehen und zu begeistern.


Wir bieten eine integrierte Komplettlösung an, welche die klassische Nutzung des Audioguides in keiner Weise beeinträchtigt. Denn jeder darf spielen, aber niemand muss!




Alles in Einem

Warum eine separate Gamification-App, wenn das Spiel in den Audioguide integriert werden kann?




Einfach zu bedienen

Die Benutzerfreundlichkeit ist äußerst intuitiv und einfach - keine langen, komplexen Anweisungen, die den Einstieg in das Spiel erschweren!

Individuell

Jede Karte hat einen einmaligen Code. Jedes Mal, wenn ein Besucher gewinnt, erhalten Sie eine E-Mail. Außerdem können Sie die Gewinner in Ihrem Kundenbereich einsehen. So können Sie Besucher individuell belohnen.

Marketing

Sie können das Spiel nutzen, um neue Besucher anzuziehen. Sie können für Ihr Café oder Geschäft als Ort werben, an dem der Preis abgeholt werden kann. Die Ergebnisse werden Material für Ihre sozialen Netzwerke generieren.



Demo-Codes anfordern


Erschwinglich

Die Entwicklung eines maßgeschneiderten Spiels ist normalerweise sehr teuer. Mit Nubart Game können Sie dies mit einer einmaligen, erschwinglichen Zahlung und ohne wiederkehrende Kosten tun.

Flexibel

Sie können wählen, ob Sie nur die geolokalisierte Widget-Suche, das Quiz mit Fragen und Antworten oder beides wünschen. Sie können auch wählen, ob Sie das Spiel für Kinder, für Erwachsene oder für beide Zielgruppen erstellen möchten.

Personalisierbar

Wir können ein individuell animiertes Widget entwerfen, dass Sie auch für weitere Werbezwecke verwenden dürfen. Des Weiteren können u.a. der Schwierigkeitsgrad, die Spiel-Stationen und die Gewinnpreise angepasst werden.

Anleitung zum Spielen mit Nubart Game

So funktioniert's

Scannen Sie den QR-Code auf der Karte

Wählen Sie dann Ihre Sprache aus dem Menü.
Wir können Nubart Game an alle Sprachen Ihres Audioguides anpassen.

Hören Sie sich die Tonspur an

Tonspuren mit Gamification sind mit der Schaltfläche “Spielen" versehen.
Sie können sie ganz einfach finden, indem Sie durch den Audioguide scrollen.

Fangen Sie das animierte Widget

Nach dem Anhören der Tonspur klicken Sie auf "Spielen", schauen Sie sich um und fangen Sie das Widget ein, das Sie in der Nähe der entsprechenden Station finden.

Beantworten Sie die Frage

Die Antwort auf die Frage ist in der Tonspur enthalten.
Wenn Sie also im Schritt 2 aufmerksam zugehört haben, werden Sie die Antwort wissen.

Wiederholen Sie dies, bis Sie gewinnen... oder verlieren.

Wiederholen Sie die obigen Schritte mit allen Tonspuren, die mit der Schaltfläche "Spielen" versehen sind. Wenn Sie eine Gewinnnachricht erhalten, können Sie Ihren Preis einfordern. Der Code auf der Karte dient als Nachweis.
Das Spiel verschwindet aus dem Audioguide, sobald es beendet wurde.
