Step 3 — Institution / venue name(s) *

Enter the name of the customer or hosting institution, plus any alternative names, venue names, or building names that may be mentioned during the event. One name per line.
Example: a museum might be known by its official name, a popular nickname, and the historic building it occupies — list all three.

Step 4 — Speaker names (optional)

Enter the full names of all speakers, one per line. These will be included in the context file to ensure correct transcription and translation of names.

Step 5 — Room / session label (optional)

For multi-room events, add a label so you can identify which context file is which. It will appear as the first line of the output.
