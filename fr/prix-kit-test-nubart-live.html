Si j'achète plusieurs paquets de base, aurai-je une licence pour plus de deux guides ?

Oui, chaque commande d'un paquet de base comprend une licence permanente pour deux guides. Si vous achetez deux forfaits, vous obtiendrez une licence pour 4 guides. Mais vous ne pourrez pas modifier ou gérer ces guides : ils sont fixes.

Dois-je acheter les 500 cartes du paquet de base même si je n'organise qu'un événement avec quelques personnes ?

Oui, le paquet de base est notre minimum de commande. Veuillez noter que les cartes n'expirent pas : vous pouvez conserver les cartes restantes pour des événements futurs.

Puis-je acheter les cartes du paquet de base et les imprimer à ma guise ?

Non. Si vous souhaitez des cartes avec votre marque, vos couleurs ou votre logo, vous devez choisir le forfait Medium ou Professional.

Les cartes ont-elles une date d'expiration ?

Non. Quelle que soit la formule choisie, vous ne devez acheter les cartes qu'une seule fois et les conserver jusqu'à ce que vous en ayez besoin. Lorsque vous n'en avez plus, vous pouvez commander des cartes de remplacement. Si vous disposez d'un package Medium et Professional, vous devrez renouveler chaque année votre licence pour les guides.
